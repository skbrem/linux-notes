## String Expressions

`test` string Expressions

| Expression | Is true if: |
| --- | --- |
| *string* | string is not null |
| `-n` *string* | The length of the string is greater than zero. |
| `-z` *string* | The length of the *string* is zero. |
| *string1* = *string2* or *string1* == *string2* | *string1* and *string2* are equal. Both single and double equal signs are usable. Although the double equal signs is not POSIX compliant, the double equal signs is more common and preferred. |
| *string1* != *string2* | *string1* and *string2* are not equal |
| *string1* > *string2* | *string1* sorts after *string2* |
| *string1* < *string2* | *string1* sorts before *string2* |
