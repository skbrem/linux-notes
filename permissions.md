The `ls` command is used to view metadata about files on Linux. 

```bash
$ ls -l
drwxr-xr-x. 4 root root    68 Jun 13 20:25 tuned
-rw-r--r--. 1 root root  4017 Feb 24  2022 vimrc
```

- File type: `-`
- Permission settings: `rw-r--r--`
- Extended attributes: `(.)`
- User owner: `root`
- Group owner: `root`

## How to Read File Permission

> [!note]
> `rw-r--r--`
